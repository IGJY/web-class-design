Sponge 用户态 TCP 协议
  • 2022.9 - 2022.10
  • 个人
  • 协议设计

Sponge 是一个在用户态实现的简易类 TCP 协议，支持滑动窗口、自动重传等机制

- 通过一个基于内存的可靠数据流和基于红黑树的重排器来提供对报文段的基本处理能力

- 根据 TCP 状态转换图实现 TCP 接收方、发送方，并对 TCP 连接进行管理

- 利用火焰图找到初版实现性能瓶颈，利用批处理思想、移动语义等进行性能优化（约为初版 9 倍）

C++ 服务器

xv6 操作系统内核功能扩展和优化
  • 2022.3 –2022.7
  • 个人
  • 内核优化

xv6 是 MIT 基于 Unix V6 开发的一个类 Unix 操作系统，在阅读 xv6 源码后，参考已有操作系统内核设计实现进行了如下功能扩展和优化

- 实现简易版的 strace、backtrace 等系统辅助工具

- 通过虚拟系统调用的方式加速只读系统调用，如：ugetpid()

- 为 fork 实现 Copy On Write 机制，优化读多写少场景下的性能与内存开销

- 优化了 xv6 文件系统 LRU 缓存以及内存分配过程中的锁竞争问题

- 为文件系统增加两级间接块，使 xv6 可以支持大文件，最大文件限制由 268KB 扩充到 65803 KB（约 245 倍）
