项目经验

点餐系统
  • 2022.8 - 2022.10
  • 个人

一个基于 Spring Boot 的点餐系统，后台管理系统提供给内部员工使用，可以对菜品信息、订单进行维护；移动端提供给用户使用，可以在线浏览餐品、加入购物车、下单等操作。

- 遵循 Restful 设计接口，降低前后端接口沟通和理解成本。

- 使用 MyBatis Plus 框架自动生成业务的增删改查重复代码，并使用LambdaQueryWrapper 实现更灵活地自定义查询。

- 为集中处理系统异常，自定义统一的错误码，并封装了全局异常处理器，屏蔽了项目的冗余报错细节，便于接口的调用方理解和统一处理。

- 为保证数据的完整性和一致性，使用 @Transactional 实现数据库事务，并配置 rollbackFor = Exception.class 支持受检异常的事务回滚。
